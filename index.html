História dos computadores

Como imagino que ninguem vai ler todo esse artigo, vou listar rapidamente algumas tecnologias dignas de nota a respeito da computação - e deixarei algumas imagens ilustrativas para sugerir que pesquisei bastante.

Ábaco (5500 a.C.), a primeira calculadora da história. Legal, né? Olhe como ela é elegante:

1638, William Oughtred criou uma tabela para a realização de multiplicações muito grandes. A base de sua invenção foram as pesquisas sobre logaritmos realizadas pelo escocês John Napier. O mecanismo consistia em uma régua que já tinha uma boa quantidade de valores pré-calculados, organizados de forma que fossem acessados automaticamente. Uma espécie de ponteiro indicava o resultado do valor desejado.

1642, o matemático francês Blaise Pascal desenvolveu o que pode ser chamado de primeira calculadora mecânica da história, a Máquina de Pascal. Seu funcionamento era baseado no uso de rodas interligadas que giravam na realização dos cálculos

1801, o costureiro Joseph Marie Jacquard desenvolveu um sistema muito interessante, a primeira máquina realmente programável. Tal mecanismo foi chamado de Tear Programável, pois aceitava cartões com entrada do sistema

1822 foi publicado um artigo científico que prometia revolucionar tudo o que existia até então no ramo do cálculo eletrônico. O autor, Charles Babbage, afirmou que a máquina era capaz de calcular funções de diversas naturezas (trigonometria, logaritmos) de forma muito simples. Esse projeto foi chamado de Máquina de Diferenças.

1837, Babbage lançou uma nova máquina, chamada de Engenho Analítico, Além disso, instruções e comandos podiam ser informados pelos cartões, fazendo uso de registradores primitivos. A precisão chegava a 50 casas decimais.

1847, o matemático George Boole criou um sistema lógico que reduzia a representação de valores com dois algarismos: 0 e 1(00: desligado; 01: carga baixa; 10: carga moderada; 11: carga alta)

1890 nos Estados Unidos, Hermann Hollerith desenvolveu uma máquina que acelerou todo o processo de computação de dados

A Segunda Guerra Mundial foi um grande incentivo no desenvolvimento de computadores, visto que as máquinas estavam se tornando mais úteis em tarefas de desencriptação de mensagens inimigas e criação de armas mais inteligentes. Entre os projetos desenvolvidos no período, os que mais se destacaram foram o Mark I, em 1944, desenvolvido na Universidade Harvard (EUA), e o Colossus, em 1946, criado por Allan Turing.

Em 1946 ocorreu uma revolução no mundo da computação com o lançamento do Electrical Numerical Integrator and Calculator (ENIAC), desenvolvido pelos cientistas norte-americanos John Eckert e John Mauchly. Essa máquina era em torno de mil vezes mais rápida que qualquer outra da época.

Como é uma estrutura básica de um HTML?

Enfim, um assunto interessante kkkk Brincadeiras a parte, vamos ententer como podemos montar uma página web em HTML

Para inicio de conversa, você pode ler o codigo dessa página apertando a tela F12. Não só dessa pagina mas de qualquer outra que você possa acessar pelos navegadores tradicionais. Vamos lá, dê uma olhada!

Perceba que na primeira linha aparece o código "< h t m l >". Ele dâ inicio à todo código feito na linguagem HTML, e lá no final das linhas de código você vai encontrar a tag "< / h t m l >", essa responsavel por marcar o fim de todo o código. Imagina essas duas tags como a proteção de capa e contra-capa de um livro, e que tudo entre elas é o site.

A orgem de tags em uma página é a seguinte:

  • < h t m l > abrindo o código
  • < h e a d > coisas que a pagina vai carregar antes de aparecer algo pro usuario, como por explo o < t i t l e > o que aparece no alto da guia < / t i t l e > < / h e a d >
  • < b o d y > o que vai aparecer ao usuario < / b o d y >
  • < / h t m l > fechando o código

E dentro desse código aplicamos varias tags e atributos conforme a necessidade do conteúdo e serviço prestado pela página

A seguir, alguns exemplos de tags e atributos disponiveis nessa linguagem:

  • italico: < i > texto a ficar em itálico < / i >
  • negrito: < s t r o n g > texto a ficar em negrito < / s t r o n g >
  • paragrafo: < p > texto dentro do paragrafo < / p >
  • citação:< b l o c k q u o t e > texto a ser citado kkk < / b l o c k q u o t e >
  • sublinhado: < u > texto a ser sublinhado < / u >
  • marca-texto: < m a r k > legal, né? fica amarelão :) < / m a r k >
  • Eu não sei o nome disso: mas a tag é < s u p > e o que você quer deixar "superior" < / s u p >
  • imagina o nome desse: < s m a l l > texto a ser diminuido < / s m a l l >
  • o contrario do < s u p >: para deixar o texto inferior é < s u b > texto < / s u b >
  • a tag < f o n t > trabalha com... fontes kkk < f o n t face="NOME DA FONTE"> Aqui é seu texto que está com a fonte "NOME DA FONTE" < / f o n t >

    exmplos:

    Arial
    Courier
    Georgia
    Helvetica
    Times
    Trebuchet
    Verdana
  • Temos esse daqui: < d e l > texto a ser... riscado? < / d e l >
  • Abreviação: ex: EUA, < a b b r title="Estados Unidos da América">EUA< / a b b r >

Entendendo a diferença entre listas ordenadas e não ordenadas

Esse é bem simples: para criar uma lista ordenada, usa-se a tag < o l > para abrir a lista, depois < l i > para listar os itens da lista < / l i >, e < / o l > para fechar a lista

exemplo:

  1. Peixe
  2. Bola
  3. Gato

E para criar uma lista não ordenada, usa-se a tag < u l > para abrir a lista, depois < l i > para listar os itens da lista < / l i >, e < / u l > para fechar a lista

exemplo:

  • Peixe
  • Bola
  • Gato

realmente não é difícil :)
